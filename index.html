Browser Security Enhancement Tracker Project

We are slowly moving our lives from desktop applications to browser based application and browsers have become an integral part of our life. Current top browsers have a special focus on security and have been working on many major security projects to kill bug classes. Some are ambitious projects some are smaller tweaks. Some continue to become the core of the browser some projects get shelved or discarded.

This brings in an interesting situation for information security professionals as our recommendations and risk ratings can get affected by these changes. We have been trying to keep a tab on these changes internally and we realized that it makes sense to make this information public. So without further ado we present you Browser Security Enhancement List
